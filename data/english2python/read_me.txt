**Usage**
In eng2c++.txt, each data entry is read line by line.
The English description and the C++ snippet are split by "\".

**Special Tokens**
"VAR": indicates all kinds of variables.
"FUNC": indicate all kinds of functions.
"INT": indicate indices, rank and etc.

**Assumptions**
STL::Vector is the assumed type for linear data structures.
  - It is as intuitive to use as generic arrays;
  - But it also supports iterators and many functions.
